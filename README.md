# Progressionist

An app that helps musicians with chords and chord progressions during song writing.

## Usage

Run `./run.py`.

## Features

Currently, the only feature is to display a musical key, by showing its chords and the notes making up those chords.

## Known Bugs

- UI prints unaligned notes when dealing with accidentals (i.e., sharps/flats); does not affect keys C Major and A Minor.
